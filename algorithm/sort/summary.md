# 排序总结

| 算法名称 |  平均时间复杂度  | 最差时间复杂度 |   最好时间复杂度   | 空间复杂度  | 是否稳定  | 是否归位  |
|:----:|:---------:|:-------:|:-----------:|:------:|:-----:|:-----:|
| 插入排序 |  O(n^2^)  | O(n^2^) |    O(n)     |  O(1)  |   是   |   否   |
| 希尔排序 | O(n^1.5^) | O(n^2^) |    O(n)     |  O(1)  |   否   |   否   |
| 选择排序 |  O(n^2^)  | O(n^2^) | **O(n^2^)** |  O(1)  |   否   |   是   |
